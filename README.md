# 🚀 Dev Workflow Automation

An intelligent developer workflow automation system built using Python.

This project eliminates repetitive manual steps in the development cycle by automatically handling Git operations, screenshot generation, and LinkedIn post preparation.

---

## 🚀 Live Automation Demonstration

This system implements a real-time automation pipeline.

Whenever a file is saved inside the project:

1️⃣ A file watcher detects the modification  
2️⃣ Changes are automatically staged and committed  
3️⃣ GitHub push is executed  
4️⃣ A timestamped project screenshot is captured  
5️⃣ A human-style LinkedIn caption is generated  
6️⃣ LinkedIn is opened with post-ready workflow  

---

## 🔄 Automation Flow

VS Code Save

## ⚙️ Technical Architecture

- `watchdog` → File system monitoring  
- `subprocess` → Git command automation  
- `mss` + `Pillow` → Screenshot engine  
- `pyperclip` → Clipboard integration  
- Modular Python project structure
